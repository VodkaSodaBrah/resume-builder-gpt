Resume Builder Widget

Embed the AI-powered resume builder on your website with just a few lines of code.

Quick Start (Automatic Initialization)

Add this snippet to your HTML to automatically display the widget:

<!-- Resume Builder Widget --> <div data-resume-builder-widget data-company-name="Your Company" data-primary-color="#22c55e" ></div> <script src="https://your-domain.com/widget/resume-builder-widget.iife.js"></script> <link rel="stylesheet" href="https://your-domain.com/widget/resume-builder-widget.css">

Manual Initialization (JavaScript API)

For more control, initialize the widget programmatically:

<script src="https://your-domain.com/widget/resume-builder-widget.iife.js"></script> <link rel="stylesheet" href="https://your-domain.com/widget/resume-builder-widget.css"> <script> // Initialize with configuration const widget = window.ResumeBuilderWidget.init({ companyName: 'Your Company', companyLogo: 'https://your-domain.com/logo.png', primaryColor: '#22c55e', allowSignup: true, defaultLanguage: 'en', apiUrl: 'https://your-api.azurewebsites.net', onComplete: function(resumeData) { console.log('Resume completed:', resumeData); // Handle completed resume }, onClose: function() { console.log('Widget closed'); } }); // Later, to destroy the widget: // window.ResumeBuilderWidget.destroy(widget); </script>

Configuration Options

Option Type Default Description
companyName string "Resume Builder" Company name displayed in widget header
companyLogo string null URL to company logo image
primaryColor string "#22c55e" Primary accent color (hex format)
allowSignup boolean true Show signup option for new users
defaultLanguage string "en" Default language code (en, es, fr, etc.)
apiUrl string "/api" Base URL for API endpoints
onComplete function null Callback when resume is completed
onClose function null Callback when widget is closed

Data Attributes (Auto-Init)

When using automatic initialization, you can set options via data attributes:

Attribute Maps To
data-company-name companyName
data-company-logo companyLogo
data-primary-color primaryColor
data-allow-signup allowSignup
data-default-language defaultLanguage
data-api-url apiUrl
data-resume-builder-config Full JSON config object

White-Label / Licensing

For white-label deployments, you can fully customize the widget appearance:

const widget = window.ResumeBuilderWidget.init({ companyName: 'Career Services Inc.', companyLogo: 'https://career-services.com/logo.png', primaryColor: '#0066cc', apiUrl: 'https://career-services-api.azurewebsites.net', allowSignup: true, onComplete: function(resumeData) { // Send to your own backend for additional processing fetch('/api/save-resume', { method: 'POST', body: JSON.stringify(resumeData) }); } });
